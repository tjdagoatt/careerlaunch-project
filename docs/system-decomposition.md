# System Decomposition

## Main Modules

### 1. Authentication Module
- User sign up
- User login/logout
- Profile management

### 2. Opportunity Search Module
- Search external job/internship listings
- View search results
- Save opportunities

### 3. Application Tracker Module
- Add applications
- Edit applications
- Delete applications
- Update status and deadlines

### 4. Skills & Development Module
- Add skill goals
- Edit progress
- Delete goals

### 5. Dashboard Module
- Show application summary
- Show deadlines
- Show skill progress
