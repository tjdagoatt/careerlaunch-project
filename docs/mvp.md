# Minimum Viable Product (MVP)

The MVP for CareerLaunch includes the following features:

- User registration and login
- Create, view, update, and delete internship/job applications
- Create, view, update, and delete skill goals
- Dashboard displaying applications and deadlines
- Opportunity search using an external API
- Save opportunities to the tracker

The MVP focuses on the smallest set of features needed to solve the core problem while remaining realistic for one semester of development.
