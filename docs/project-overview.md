# Project Overview

## Project Title
CareerLaunch

## Topic
Career Preparation & Professional Development

## Summary
CareerLaunch is a full-stack web application that helps students and recent graduates manage career preparation activities in one place. Users can search for internship and job opportunities, save applications, track deadlines, monitor progress, and manage skill development goals. The application addresses the common problem of disorganized career planning by centralizing important information into a single platform.
